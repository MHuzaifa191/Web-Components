### Installation and Usage Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MHuzaifa191/React-Components.git
   cd React-Components
   cd Project-Folder # e.g. Notes App
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the application:**
   ```bash
   npm start
   ```
